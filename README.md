# ayat-mushaf-assets

Static mushaf page images for the **Ayat Shorts** app, served via the free jsDelivr CDN.

- `jawami/pageNNN.webp` — coloured Jawami'i print (604 pages)
- `tajweed/pageNNN.webp` — coloured tajweed print (604 pages)

Pages follow the standard 604-page Madani order. Referenced by the app at
`https://cdn.jsdelivr.net/gh/husam550-ship-it/ayat-mushaf-assets@v1/<edition>/pageNNN.webp`.
